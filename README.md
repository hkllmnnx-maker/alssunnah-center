# مركز السنة للعلوم الشرعية والتأهيل

## نظرة عامة على المشروع
- **الاسم**: مركز السنة للعلوم الشرعية والتأهيل (alssunnah-center)
- **الهدف**: موقع رسمي للمركز يقدّم رحلة علمية متكاملة لطالب العلم، بهوية بصرية فاخرة (Deep Emerald + Royal Gold) وتجربة استخدام عصرية.
- **اللغة**: العربية (RTL)
- **الهوية البصرية**: زمردي عميق `#062E25` + ذهبي ملكي `#C5A059`

## المميزات المنفّذة (Phase 1 + Phase 2)
- ✅ **Premium Navbar**: شريط علوي رخامي + شعار + قائمة 6 عناصر + زر الوضع الليلي + CTA "انضم إلينا".
- ✅ **Hero Section فاخر**: خلفية زمردية عميقة + زخارف إسلامية بشفافية 5% + حلقات دوّارة + شعار المركز.
- ✅ **بطاقات القيم الأساسية**: الرؤية / الرسالة / الأهداف بتأثير ارتفاع عند المرور (`soft-card`).
- ✅ **الأقسام العلمية**: شبكة من ست أقسام (العقيدة، التفسير، الحديث، الفقه، اللغة، السيرة) بأيقونات ذهبية.
- ✅ **المكتبة الرقمية**: عرض كاروسيل (Swiper) + شبكة كاملة بفلتر تصنيفات + نافذة معاينة كل كتاب.
- ✅ **معرض الوسائط**: تبويبات (مرئيات / صوتيات) + مشغل بارز + شبكة استجابية مع فلاتر.
- ✅ **التذييل الراقي**: 4 أعمدة + اشتراك بالنشرة + أيقونات اجتماعية (فيسبوك / تليجرام / يوتيوب / واتساب).
- ✅ **الوضع الليلي**: مفعّل عبر `class` على `<html>` + حفظ في `localStorage` + سكريبت `FOUC-prevention`.
- ✅ **RTL Native**: `dir="rtl"` + خطوط عربية (Cairo / Tajawal / Amiri).
- ✅ **AOS Animations**: حركات سلسة عند التمرير على كل الأقسام.
- ✅ **Forms + APIs**: نموذج تسجيل + نموذج تواصل + مدوّنة 404 مخصصة.

## مسارات الموقع (Routes)
| المسار | الوصف |
|--------|-------|
| `/` | الصفحة الرئيسية (8 أقسام بمعمارية Phase 2) |
| `/about` | عن المركز (الرؤية، الرسالة، القيم، الأهداف، المنهجية) |
| `/departments` | **الأقسام العلمية** (تفصيل 6 أقسام مع المقررات والمحاور) |
| `/library` | **المكتبة الرقمية** (كاروسيل + شبكة 14 كتاباً + Modal PDF) |
| `/media` | **الوسائط** (مرئيات + صوتيات + فلاتر + مشغل) |
| `/courses` | الدورات والبرامج (الماهر 3 + 6 دورات + فلتر) |
| `/gallery` | معرض الصور |
| `/news` | الأخبار |
| `/contact` | تواصل معنا |
| `/register` | التسجيل في الدورات |

## نقاط API
| المسار | الطريقة | الوصف |
|--------|---------|-------|
| `/api/register` | POST | استقبال طلبات التسجيل (يتطلب `fullName`, `phone`, `course`) |
| `/api/contact` | POST | استقبال رسائل التواصل (يتطلب `name`, `phone`, `message`) |

## معمارية البيانات
- **النماذج**:
  - `RegisterRequest`: { fullName, phone, email?, age?, course, notes? }
  - `ContactRequest`: { name, phone, email?, subject?, message }
- **التخزين الحالي**: log إلى console (للمراحل التطويرية).
- **التخزين المخطط**: Cloudflare D1 / KV (يمكن إضافته بسهولة لاحقاً).

## التقنيات المستخدمة
- **Backend**: [Hono](https://hono.dev/) + JSX renderer
- **Build**: Vite 6 + @hono/vite-build
- **Deployment**: Cloudflare Pages (`wrangler pages dev/deploy`)
- **Styling**: Tailwind CSS (CDN) مع توسعة الألوان (`emerald-deep`, `royal-gold`)
- **Components**: Swiper.js v11 + AOS + Font Awesome 6
- **Fonts**: Google Fonts (Cairo, Tajawal, Amiri)
- **Process manager**: PM2 (`ecosystem.config.cjs`)

## كيفية تشغيل المشروع محليًا
```bash
# تثبيت الحزم (مرة واحدة)
npm install

# بناء المشروع
npm run build

# تشغيل عبر PM2 (محلي/Sandbox)
pm2 start ecosystem.config.cjs

# اختبار سريع
curl http://localhost:3000
```

## النشر إلى Cloudflare Pages
```bash
npm run build
npx wrangler pages deploy dist --project-name <cloudflare_project_name>
```

## دليل الاستخدام
1. الصفحة الرئيسية تعرض هوية المركز ورؤيته وأقسامه ومختاراته.
2. زر **"انضم إلينا"** في الشريط العلوي يأخذك مباشرة لصفحة التسجيل.
3. زر **القمر/الشمس** يبدّل الوضع الليلي ويُحفظ في المتصفح.
4. صفحة **الأقسام العلمية** تعرّفك بكل قسم: المقررات، المحاور، والمنهجية.
5. **المكتبة الرقمية** تتيح تصفية الكتب حسب التصنيف ومعاينة تفاصيلها قبل التحميل.
6. **الوسائط** تجمع المرئيات والصوتيات في واجهة موحدة بتبويبات.
7. **تواصل معنا** و **التسجيل** يقدمان نماذج متصلة بـ API.
